Automatic Dog Feeder (2021)

The project involved the design and construction of an automatic dog feeder for my dog. The feeder was designed to dispense exactly 2 cups of dog food at set intervals, providing a convenient and reliable way to feed my pet. To create the feeder, I purchased a housing for storing the dog food and modified it by attaching a servo motor to the turn valve of the dispenser. I then used an Arduino Uno to control the operation of the servo motor, carefully calibrating the timings to dispense the exact amount of food. One of the challenges of the project was getting the timings for the servo operation just right. I had to experiment with different durations and fine-tune the settings to ensure that the correct amount of food was dispensed each time. Another challenge was attaching the servo to the twist knob of the dispenser, as it required some mechanical ingenuity to securely mount the motor while still allowing it to operate smoothly. In the end, the project was a success, and I was able to create a fully functional automatic dog feeder that dispensed the correct amount of food at the desired intervals. The feeder has been in use for several months now, and it has proved to be a convenient and reliable way to ensure that my dog is well-fed.

DoodleBot (2021)

DoodleBot is a simple robot that can draw basic shapes using a 3D printed skeleton, two motors with wheels, and a pen. The robot is powered by an Elegoo UNO R3 microcontroller, which enables it to move in a pre-programmed pattern based on the code that is used for its motor function. The user can input the quantity and pattern of steps for the motors, which determines the shape that the robot will draw. For example, if the user inputs a pattern of four steps forward and two steps back, the robot will move in a square pattern, drawing a square shape on the surface below.

NaviBot (2022)

NaviBot is a simple robot that uses ultrasonic sensors to navigate around obstacles. The robot is partially constructed using the same 3D printed skeleton as DoodleBot, but it has no pen attached and is equipped with two mounted 40kHz ultrasonic sensors on the front. For example, when the ultrasonic sensors emit waves that bounce back, indicating the presence of an object in front of the robot, the two motors will stop and the right motor will turn NaviBot until the ultrasonic sensors no longer detect any objects. This allowed the robot to navigate around obstacles without colliding with them.

GamingPi (2022)

The Retro Gaming Console project is a simple and fun project that uses a Raspberry Pi Zero W, a NES USB controller, and an emulator software called RetroPie to create a retro gaming console. The project involves installing RetroPie on the Raspberry Pi Zero W, connecting the NES USB controller, and configuring the emulator to play classic games from the NES. Revisting the classics with this project was something I just had to try, plus it initially got me familiar with some of the Raspberry Pi's functions.
